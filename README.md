TravelWeather Pro
Aplicación interactiva de clima y recomendaciones de viaje desarrollada por Daniel Burgos para la clase de Diseño Front-End.

Descripción
Dashboard que muestra el clima actual, pronóstico de 5 días, calidad del aire y sugerencias dinámicas de comida y ropa basadas en la temperatura.

Funcionalidades
-Clima actual y pronóstico extendido por horas.
-Indicador de Calidad del Aire (AQI).
-Recomendaciones de gastronomía y vestimenta según el clima.
-Sistema de favoritos con almacenamiento local (localStorage).
-Mapas interactivos de temperatura.

Tecnologías
-HTML5 y CSS3 (Tailwind CSS).
-JavaScript Vanilla.
-API OpenWeatherMap.
-Leaflet.js [cite: 2026-02-16].
